# Paylink – M-Pesa Payment Links

Paylink is a full-stack payment link system that allows businesses to create shareable payment links, collect payments via M-Pesa STK push, and automatically generate PDF receipts. Built for the Kenyan market, it simulates a real-world fintech integration using Safaricom's Daraja API.

---

## Features

- Create payment links – dashboard with client name, phone, amount, description.
- M-Pesa STK push – initiates a payment prompt on the customer's phone.
- Asynchronous webhooks – Safaricom calls your callback URL when payment is completed.
- Amount validation – compares the amount received with the amount requested; rejects mismatches.
- Idempotent callbacks – duplicate callbacks are ignored (prevents double processing).
- PDF receipt generation – downloadable receipt with transaction details (amount, receipt number, date).
- Polling frontend – payment page updates in real time while waiting for confirmation.
- Security – secrets stored in .env, no money-critical code generated by AI.
- Persistence – SQLite for payment links, JSON file store for transaction intents (sandbox reconciliation).
